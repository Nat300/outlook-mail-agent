# Outlook Mail Agent

A learning project — an AI-powered agent that connects to my Outlook inbox and automatically classifies emails into categories like spam, newsletters, action required, and important.

Built to deepen my understanding of AI agents, APIs, and automation. Everything is documented step by step as I go.

---

## Goal

Connect to Outlook → read emails → classify them with Claude → move them into the right folders automatically.

---

## Project Status

Work in progress — just getting started
